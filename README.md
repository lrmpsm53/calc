# Расшираемый (почти) калькулятор

## Установка
+ Установить NodeJS
+ `git clone git@github.com:lrmpsm53/excalc.git`
+ `npm install`
+ `npm run start`
## Глобальная устновка (Linux)
+ `sudo npm -g install ./`
+ `calc`

## Доступные операторы
`+, -, *, /, cos, sin, and, or, not`
